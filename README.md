# vesta.social

Static landing page for [vesta.social](https://vesta.social).

## Files

- `index.html` — the page
- `tree.svg` — hero artwork
- `CNAME` — custom domain for GitHub Pages
- `favicon.svg`
- `robots.txt`

## Local preview

Open `index.html` in any browser.

## Deploy

Pushed to `main` is served automatically by GitHub Pages.
